1. About Luelite

Luelite is a custom fork of Lua 5.5.0, designed specifically for the Retrelite game engine. It combines the simplicity of Lua with unique features for game development and local AI integration.

Version 0.1.0 (Prototype)
Based on Lua 5.5.0
Author Elite
License MIT

Note: Luelite is currently in prototype stage. Some features are marked as TODO and will be implemented in future versions.

2. Features Overview

Feature Status Description
wait(seconds) DONE Yields current coroutine for N seconds. Default: 1/30s.
spawn(function) DONE Runs function in a new coroutine. Non-blocking.
delay(seconds, function) DONE Calls function after N seconds. Implemented in Lua.
tick() DONE Returns global engine time in seconds.
match(value, {}) DONE Pattern matching. Cleaner than if/elseif chains.
http.get(url) DONE HTTP GET requests. Direct access to localhost APIs.
http.post(url, body) TODO HTTP POST for Ollama API integration.
vector3 TODO 3D vector type with operators.
signal TODO Event system for game objects.
print_r() TODO Recursive table printer for debugging.

3. Installation & Usage

3.1 Building from Source

git clone https://github.com/yourname/luelite.git
cd luelite
mkdir build && cd build
cmake .. -G "MinGW Makefiles"
make

3.2 Adding to PATH

Copy luelite.exe to a folder in your PATH (e.g. C:\Users\You\retrelite\).

3.3 Running Scripts

luelite myscript.lua
luelite myscript.luelite

Supported extensions: .lua and .luelite

3.4 File Association (Windows)

Run associate_luelite.bat as Administrator to associate .luelite files with Luelite.

4. Multitasking

Luelite supports cooperative multitasking through coroutines. This allows multiple scripts to run "simultaneously" without blocking each other.

4.1 wait()

-- Yields for 0.5 seconds
wait(0.5)
print("Half a second passed!")

4.2 spawn()

-- Runs in background
spawn(function()
    for i = 1, 5 do
        print("[BG] " .. i)
        wait(0.3)
    end
end)

-- Main thread continues immediately
print("[Main] Doing other work...")
wait(2)
print("[Main] Done!")

Output:

[Main] Doing other work...
[BG] 1
[BG] 2
[BG] 3
[BG] 4
[BG] 5
[Main] Done!

4.3 delay()

delay(1.0, function()
    print("1 second delayed message")
end)

delay(0.2, function()
    print("0.2 second delayed message")
end)

wait(1.5)

Output:

0.2 second delayed message
1 second delayed message

4.4 tick()

print("Start time: " .. tick())
wait(1)
print("Current time: " .. tick())
print("Elapsed: ~1 second")

5. Pattern Matching (match)

match is a powerful replacement for long if/elseif/else chains. Inspired by pattern matching in Rust and Haskell.

5.1 Basic Syntax

-- Syntax
match(value, {
    [pattern1] = result1,
    [pattern2] = result2,
    [_default()] = fallback_result
})

5.2 Numeric Matching

local health = 75

local status = match(health, {
    [_lt(0)] = "dead",
    [0] = "dead",
    [_between(1, 25)] = "critical",
    [_between(26, 50)] = "wounded",
    [_between(51, 99)] = "fine",
    [_default()] = "perfect"
})

print(status) -- "fine"

5.3 String Matching

local command = "jump"

match(command, {
    ["walk"] = function() print("Walking...") end,
    ["jump"] = function() print("Jumping!") end,
    ["run"] = function() print("Running!") end,
    [_default()] = function() print("Idle") end
})

5.4 Match Markers Reference

Marker Description Example
_lt(n) Matches if value < n [_lt(0)] = "negative"
_between(a, b) Matches if a ≤ value ≤ b [_between(1, 10)] = "small"
_default() Fallback when nothing matches [_default()] = "other"
exact value Matches exact value [42] = "answer"
Note: Markers currently work with integers in range -100 to 100. This covers most game use cases (health, scores, levels, states).

6. HTTP Client

Luelite includes a built-in HTTP client for direct API access. No external dependencies required.

6.1 http.get()

local body, err = http.get("http://example.com")

if body then
    print("Response: " .. #body .. " bytes")
    print(body:sub(1, 100))
else
    print("Error: " .. err)
end

6.2 Ollama Integration

-- Check if Ollama is running
local response, err = http.get("http://localhost:11434/api/tags")

if response then
    print("Ollama is online!")
    -- Parse response...
else
    print("Ollama not available")
end
Important: http.post is not yet implemented. Coming in 0.2.0 for full Ollama API support.

8. Complete Example

A script that uses all Luelite features together:

-- combo_test.luelite
io.stdout:setvbuf("no")

print("=== LUELITE COMBO TEST ===\n")
print("Tick: " .. tick())

-- Multitasking
spawn(function()
    for i = 1, 3 do
        print("  [BG] " .. i)
        wait(0.3)
    end
    print("  [BG] done!")
end)

spawn(function()
    wait(0.15)
    print("  [Fast] done!")
end)

for i = 1, 2 do
    print("  [Main] " .. i)
    wait(0.2)
end
wait(1.2)

-- Delay
delay(0.3, function() print("  [Delay 0.3] fired!") end)
delay(0.1, function() print("  [Delay 0.1] fired!") end)
wait(0.5)

-- Match
local hp = 75
local status = match(hp, {
    [_lt(0)] = "dead",
    [0] = "dead",
    [_between(1, 25)] = "critical",
    [_between(26, 50)] = "wounded",
    [_between(51, 99)] = "fine",
    [_default()] = "perfect"
})
print("HP: " .. hp .. " -> " .. status)

match("attack", {
    ["idle"] = function() print("State: standing") end,
    ["attack"] = function() print("State: attacking!") end,
    [_default()] = function() print("State: unknown") end
})

-- HTTP
local body, err = http.get("http://example.com")
if body then
    print("HTTP: online (" .. #body .. " bytes)")
else
    print("HTTP: offline")
end

print("\nFinal tick: " .. tick())
print("=== ALL TESTS PASSED ===")
